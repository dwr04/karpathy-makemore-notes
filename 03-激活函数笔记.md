# 激活函数全解:图像 · 导数 · 场景 · 优缺点 · 梯度爆炸/消失

> 承接 `02-数据流与梯度流笔记.md`:你已经见过 tanh 饱和 → 反向梯度死亡。
> 这篇把主流激活函数横向铺开,核心抓手依然是那句话:
> **激活函数的导数,决定了梯度能不能活着传回去。**

---

## 0. 为什么需要激活函数(一句话)

没有它,`W2(W1·x)` 无论堆多少层都还是一个线性变换,等价于单层,学不了非线性关系。
激活函数提供**非线性**,让网络能拟合复杂函数。代价是:它的导数会缩放梯度,
用不好就引发梯度消失/爆炸。

---

## 1. 速查总表

| 函数 | 公式 | 输出范围 | 导数范围 | 零中心 | 饱和? | 主要场景 |
|---|---|---|---|---|---|---|
| **Sigmoid** | 1/(1+e⁻ˣ) | (0,1) | (0, 0.25] | ❌ | 两端 | 二分类输出、门控 |
| **Tanh** | (eˣ-e⁻ˣ)/(eˣ+e⁻ˣ) | (-1,1) | (0, 1] | ✅ | 两端 | RNN、浅层隐藏层 |
| **ReLU** | max(0,x) | [0,∞) | {0,1} | ❌ | 负半轴死 | CNN/MLP 默认首选 |
| **Leaky ReLU** | max(αx,x) | (-∞,∞) | {α,1} | ≈✅ | 不饱和 | 修 dead ReLU |
| **PReLU** | max(αx,x), α可学 | (-∞,∞) | {α,1} | ≈✅ | 不饱和 | α 让网络自己学 |
| **ELU** | x / α(eˣ-1) | (-α,∞) | (0,1] | ≈✅ | 负端软饱和 | 想要负值+平滑 |
| **GELU** | x·Φ(x) | ≈(-0.17,∞) | 平滑 | ≈✅ | 平滑 | Transformer 标配 |
| **Swish/SiLU** | x·sigmoid(βx) | ≈(-0.28,∞) | 平滑 | ≈✅ | 平滑 | 深层网络、替代 ReLU |
| **Softmax** | eˣⁱ/Σeˣ | (0,1) 且和=1 | 见正文 | — | — | 多分类输出层 |

> "零中心"=输出均值是否≈0。不零中心(sigmoid、relu)会让下一层输入恒正,
> 梯度更新偏向一个方向,收敛更慢(下面 sigmoid 缺点会细讲)。

## 2. Sigmoid

```
        1 |          ......------
          |      ...
      0.5 |----.------------------  σ(x)=1/(1+e^-x)
          |  ..
        0 |----------------
              x=0
导数: σ'(x) = σ(x)(1-σ(x))     最大值在 x=0 处 = 0.25
```

**图像**:S 形,把整个实数轴压到 (0,1)。
**导数**:钟形,峰值只有 **0.25**,两端趋于 0。

**优点**
- 输出可解释为"概率"(0~1),适合二分类最后一层、以及 LSTM/GRU 的门控。

**缺点(三宗罪)**
1. **饱和杀梯度**:|x| 稍大导数就≈0,深层网络里梯度连乘迅速消失。
2. **导数峰值只有 0.25**:即使不饱和,每过一层梯度至少 ×0.25 → 天然衰减。
3. **非零中心**:输出恒正 → 下一层权重梯度同号,走"之"字形,收敛慢。

**场景**:现在基本只用在**输出层做二分类概率**、或**门控机制**;隐藏层几乎不用了。

## 3. Tanh(你在 makemore 里用的)

```
        1 |        .....-------
          |     ..
        0 |----.--------------   tanh(x)
          |  ..
       -1 |...----------------
              x=0
导数: tanh'(x) = 1 - tanh²(x)    最大值在 x=0 处 = 1
```

**图像**:和 sigmoid 同为 S 形,但压到 **(-1,1)**,**零中心**。
**导数**:峰值 **1**(比 sigmoid 好 4 倍),但两端同样饱和到 0。

**优点**
- **零中心**:比 sigmoid 收敛快,这是它取代 sigmoid 做隐藏层的主因。
- 导数峰值 1,梯度衰减比 sigmoid 温和。

**缺点**
- **仍然两端饱和** → 深层还是会梯度消失。
- 你自己实测过:naive 初始化让 tanh 饱和 64%,`1-a1²` 均值掉到 0.13,梯度只剩 13%。

**场景**:RNN 隐藏状态、浅层 MLP(如本项目)。
**关键联系**:`数据流笔记` 里的 `dz1 = da1*(1-a1²)`,那个 `1-a1²` 就是这里的导数。
控制饱和(好初始化 / BatchNorm)= 保住这个导数 = 保住梯度。

## 4. ReLU(现代默认)

```
        |        /
        |       /
      0 |______/________   ReLU(x)=max(0,x)
        |  x<0 恒0
导数: 1 (x>0) / 0 (x<0)    —— 正半轴梯度恒为 1,不衰减!
```

**图像**:负的全砍成 0,正的原样通过。
**导数**:非 0 即 1,**正半轴恒 1**——这是它对抗梯度消失的杀手锏。

**优点**
1. **正区不饱和**:梯度恒为 1,连乘也不衰减 → 能训很深的网络。
2. **计算极快**:一个 max,没有 exp。
3. **稀疏激活**:约一半神经元输出 0,天然稀疏,某种正则效果。

**缺点:Dead ReLU(神经元死亡)**
- 若某神经元输入长期为负,输出恒 0、导数恒 0 → **梯度永远为 0,再也更新不了**,彻底死掉。
- 诱因:学习率太大导致一次大更新把权重推到"永远负"的区域;或初始化不当。
- 缓解:用较小学习率、He 初始化、或换下面的 Leaky/ELU。
- **非零中心**(输出恒 ≥0),有 sigmoid 那种"之"字问题,但因为其他优点太强,可接受。

**场景**:CNN、MLP 的**默认隐藏层激活**,不知道选啥就先用它。

## 5. ReLU 的改良家族

### 5.1 Leaky ReLU
```
f(x) = x (x>0) / αx (x≤0),  α 通常 0.01
导数 = 1 (x>0) / α (x≤0)    —— 负半轴留一条小缝,梯度不为0
```
- **解决 dead ReLU**:负区导数是 α 而非 0,神经元有机会"复活"。
- 缺点:α 是拍脑袋定的超参数。

### 5.2 PReLU
- 把 Leaky 的 α 变成**可学习参数**,让网络自己决定负区斜率。
- 更灵活,但多了参数、可能过拟合小数据集。

### 5.3 ELU
```
f(x) = x (x>0) / α(e^x - 1) (x≤0)
```
- 负区是平滑曲线,**输出可为负 → 更接近零中心**,收敛更稳。
- 负端**软饱和**(趋于 -α),对噪声更鲁棒。
- 缺点:负区要算 exp,比 ReLU 慢。

---

## 6. 平滑现代派:GELU / Swish

### 6.1 GELU(Transformer 标配)
```
GELU(x) = x · Φ(x)     Φ 是标准正态的累积分布函数(CDF)
        ≈ x · sigmoid(1.702x)   (近似式)
```
- 直觉:不像 ReLU 硬砍,而是"按输入大小的概率"平滑地保留 → 处处可导、无死区。
- 允许小的负值通过,梯度更平滑。
- **场景**:BERT、GPT 等几乎所有 Transformer 的 FFN 层。

### 6.2 Swish / SiLU
```
Swish(x) = x · sigmoid(βx)     β=1 时即 SiLU
```
- 谷歌搜索得到的函数,**非单调**(负区有个小凹陷),深层网络上常略胜 ReLU。
- 和 GELU 形状很像,都是"平滑版 ReLU"。
- **场景**:EfficientNet、部分现代 CNN 和 LLM。

> **趋势**:硬拐角的 ReLU → 平滑的 GELU/Swish。平滑意味着处处有梯度、无死区,
> 在很深的网络里训练更稳。代价是计算稍贵。

## 7. Softmax(输出层专用,不是隐藏层激活)

```
softmax(x)_i = e^{x_i} / Σ_j e^{x_j}     输出是概率分布(全正、和为1)
```
- **不是**逐元素激活,而是把一整个向量归一化成概率分布。
- **只用在多分类输出层**,配合交叉熵。你在 makemore 里的 `probs` 就是它。
- 数值技巧(见全流程笔记):先减每行最大值防 `exp` 溢出。
- 和 CE 合并求导极简:`dlogits = probs - onehot(y)`。

## 8. 梯度消失 vs 梯度爆炸(把上面全串起来)

### 8.1 根源:反向传播是"连乘"

L 层网络,输入层梯度 ≈ 各层"权重 × 激活导数"的**连乘**:
```
dLoss/d(浅层) ≈ Π (W_i^T · f'(·))     —— L 个因子相乘
```
每个因子若普遍 <1 → 连乘指数级 →0(**消失**);若普遍 >1 → 指数级 →∞(**爆炸**)。

### 8.2 梯度消失

| 成因 | 说明 |
|---|---|
| **激活饱和** | sigmoid/tanh 两端导数→0,连乘迅速归零(你实测过 naive tanh 只剩 13%) |
| **sigmoid 导数上限 0.25** | 就算不饱和,每层至少 ×0.25,10 层就是 0.25¹⁰≈1e-6 |
| **权重初始化过小** | `std(W)` 小 → 前向信号塌缩、反向梯度也塌缩(你实测 `*0.1` → std 0.54) |

**后果**:浅层(靠近输入)几乎不更新,网络"学不动前面"。

### 8.3 梯度爆炸

| 成因 | 说明 |
|---|---|
| **权重初始化过大** | `std(W)` 大 → 每层放大(你实测 naive → hprebn std 5.79) |
| **网络太深 + 无归一化** | 微小放大被连乘指数放大 |
| **RNN 处理长序列** | 同一权重沿时间反复连乘,最易爆炸 |

**后果**:梯度变 inf/NaN,loss 突然飙升或变 NaN,训练崩溃。

### 8.4 六大解药(按重要性)

1. **好的初始化**(Xavier/Kaiming):让 `std(W)=gain/√fan_in`,前向后向都保 std≈1。← 你已掌握
2. **归一化层**(BatchNorm/LayerNorm):每层强制把分布拽回 N(0,1),从根上防饱和/爆炸。← 02-3 用了
3. **不饱和激活**(ReLU 系):正区导数恒 1,不引入 <1 的衰减因子。
4. **残差连接**(ResNet 的 skip):`y=x+f(x)`,给梯度一条"高速公路"直达浅层,深网必备。
5. **梯度裁剪**(gradient clipping):`if ||grad||>阈值: 缩放`,专治**爆炸**(RNN 常用)。
6. **合适学习率**:过大既会 dead ReLU 也会诱发爆炸。

### 8.5 一句话对照

| | 梯度消失 | 梯度爆炸 |
|---|---|---|
| 现象 | 浅层不更新、loss 卡住不降 | loss 飙升/变 NaN |
| 主因 | 饱和激活 + 小权重 + 太深 | 大权重 + 太深 + RNN 长序列 |
| 主药 | 好初始化 + 归一化 + ReLU + 残差 | 好初始化 + 归一化 + 梯度裁剪 |

## 9. 实战选型建议

```
隐藏层默认           →  ReLU(简单快、够用)
Transformer/大模型   →  GELU(或 Swish)
遇到 dead ReLU       →  Leaky ReLU / ELU
RNN 循环单元         →  tanh(状态) + sigmoid(门控)
二分类输出           →  Sigmoid
多分类输出           →  Softmax
```

**给你的项目**:makemore 的 tanh 是教学选择(方便手推 `1-a1²`)。
真要提性能,可把隐藏层换 ReLU + Kaiming(relu) 初始化,饱和问题直接消失。

## 10. 一句话总结

- **激活函数=非线性来源**,但它的**导数**同时决定梯度存亡。
- **饱和型**(sigmoid/tanh):两端导数→0,深层易梯度消失;tanh 因零中心优于 sigmoid。
- **ReLU 系**:正区导数恒 1,不饱和、能训深网,代价是 dead ReLU 和非零中心;Leaky/ELU 来补。
- **平滑派**(GELU/Swish):处处可导无死区,Transformer 时代主流。
- **梯度消失/爆炸的本质是连乘**:消失=因子普遍<1(饱和/小权重),爆炸=因子普遍>1(大权重)。
- **通用解药**:好初始化 + 归一化 + 不饱和激活 + 残差连接;爆炸再加梯度裁剪。
